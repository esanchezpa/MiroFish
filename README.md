## ⚡ Descripción General

**MiroFish** es un motor de predicción de IA de nueva generación impulsado por tecnología multi-agente. A partir de semillas de información del mundo real (como noticias de última hora, borradores de políticas o señales financieras), construye automáticamente un mundo digital paralelo de alta fidelidad. En este espacio, miles de agentes inteligentes con personalidades independientes, memoria a largo plazo y lógica de comportamiento interactúan libremente y experimentan evolución social. Puedes inyectar variables dinámicamente desde una "vista de Dios" para deducir con precisión trayectorias futuras — **ensaya el futuro en un entorno digital y gana decisiones tras incontables simulaciones**.

> Solo necesitas: Subir materiales semilla (informes de análisis de datos o historias interesantes) y describir tus requisitos de predicción en lenguaje natural</br>
> MiroFish devolverá: Un informe de predicción detallado y un mundo digital de alta fidelidad con el que podrás interactuar profundamente.

### Nuestra Visión

MiroFish está dedicado a crear un espejo de inteligencia de enjambre que mapea la realidad. Al capturar la emergencia colectiva desencadenada por las interacciones individuales, superamos las limitaciones de la predicción tradicional:

- **A nivel macro**: Somos un laboratorio de ensayo para los tomadores de decisiones, permitiendo que las políticas y las relaciones públicas sean probadas con riesgo cero
- **A nivel micro**: Somos un entorno creativo para usuarios individuales — ya sea deduciendo finales de novelas o explorando escenarios imaginativos, todo puede ser divertido, lúdico y accesible

Desde predicciones serias hasta simulaciones entretenidas, dejamos que cada "¿qué pasaría si?" vea su resultado, haciendo posible predecir cualquier cosa.

## 🔄 Flujo de Trabajo

1. **Construcción del Grafo**: Extracción de semillas & Inyección de memoria individual/colectiva & Construcción de GraphRAG
2. **Configuración del Entorno**: Extracción de relaciones entre entidades & Generación de perfiles & Inyección de configuración de agentes
3. **Simulación**: Simulación paralela en doble plataforma & Análisis automático de requisitos de predicción & Actualizaciones dinámicas de memoria temporal
4. **Generación de Informe**: ReportAgent con un rico conjunto de herramientas para interacción profunda con el entorno post-simulación
5. **Interacción Profunda**: Chat con cualquier agente en el mundo simulado & Interactuar con ReportAgent

## 🚀 Inicio Rápido

### Opción 1: Despliegue desde Código Fuente (Recomendado)

#### Requisitos Previos

| Herramienta | Versión | Descripción | Verificar Instalación |
|-------------|---------|-------------|----------------------|
| **Node.js** | 18+ | Runtime de frontend, incluye npm | `node -v` |
| **Python** | ≥3.11, ≤3.12 | Runtime de backend | `python --version` |
| **uv** | Última | Gestor de paquetes Python | `uv --version` |

#### 1. Configurar Variables de Entorno

```bash
# Copiar el archivo de configuración de ejemplo
cp .env.example .env

# Editar el archivo .env y completar las claves API requeridas
```

**Variables de Entorno Requeridas:**

```env
# Configuración de API LLM (soporta cualquier API LLM con formato OpenAI SDK)
# Recomendado: modelo Alibaba Qwen-plus vía Plataforma Bailian: https://bailian.console.aliyun.com/
# Alto consumo, prueba primero simulaciones con menos de 40 rondas
LLM_API_KEY=tu_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus

# Configuración de Zep Cloud
# La cuota mensual gratuita es suficiente para uso simple: https://app.getzep.com/
ZEP_API_KEY=tu_zep_api_key
```

#### 2. Instalar Dependencias

```bash
# Instalación en un clic de todas las dependencias (raíz + frontend + backend)
npm run setup:all
```

O instalar paso a paso:

```bash
# Instalar dependencias de Node (raíz + frontend)
npm run setup

# Instalar dependencias de Python (backend, crea entorno virtual automáticamente)
npm run setup:backend
```

#### 3. Iniciar Servicios

```bash
# Iniciar tanto frontend como backend (ejecutar desde la raíz del proyecto)
npm run dev
```

**URLs de los Servicios:**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:5001`

**Iniciar Individualmente:**

```bash
npm run backend   # Iniciar solo el backend
npm run frontend  # Iniciar solo el frontend
```

### Opción 2: Despliegue con Docker

```bash
# 1. Configurar variables de entorno (igual que el despliegue desde fuente)
cp .env.example .env

# 2. Descargar imagen e iniciar
docker compose up -d
```

Lee `.env` del directorio raíz por defecto, mapea los puertos `3000 (frontend) / 5001 (backend)`

> La dirección del espejo para una descarga más rápida se proporciona como comentario en `docker-compose.yml`, reemplazar si es necesario.
